# Portfolio

This repository contains my personal portfolio.

Requires Docker and Docker-Compose to be installed and is meant to be used within a Linux machine.

In order to start the containers, execute the command "docker-compose up" within the root folder.

You can access the website by entering '0.0.0.0:8000' in the web browser of your choice while it is active within your machine.

This is a work in progress.
